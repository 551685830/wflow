# 概述

IOT解析平台 是一个通用型基础物联网云平台，与阿里云iot、onenet、涂鸦智能等平台类似，用户只需要按照标准，即可通过各类方式将任意品类的设备连入本平台，也可以无需开发，仅通过简单的配置即可将本平台内的所有设备接入其他系统平台，大大减少开发工作量，为企业节省人力物力成本。
